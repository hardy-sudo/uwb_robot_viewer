====================================================
  RCS 로봇 제어 프로그램 - 전달 패키지
  (PAUSE / RESUME / CHARGE)
====================================================

[ 포함 파일 ]

  RobotControl.exe     PC용 실행파일 (Python 설치 불필요)
  robot_control.html   스마트폰 웹앱
  proxy_server.py      스마트폰용 프록시 서버
  접속_정보.txt         IP / Port / API 정보 요약
  통신_확인.docx        API 명세 문서

----------------------------------------------------

[ 방법 1 ] PC에서 바로 실행 (가장 간단)

  1. RobotControl.exe 더블클릭
  2. 상단에서 IP / PORT / ROBOT ID 확인
     - IP      : 10.0.4.104
     - PORT    : 7000
     - ROBOT ID: 0001
  3. PAUSE / RESUME / CHARGE 버튼으로 제어

  ※ Python 설치 없이 바로 실행 가능
  ※ RCS 서버와 같은 내부망 연결 필요

----------------------------------------------------

[ 방법 2 ] 스마트폰으로 제어 (Wi-Fi 필요)

  [PC에서]
  1. Python 설치 확인 (없으면 python.org 에서 설치)
  2. proxy_server.py 실행
     > python proxy_server.py
  3. 화면에 표시된 접속 주소 확인
     예) http://192.168.1.10:8080

  [스마트폰에서]
  4. PC와 같은 Wi-Fi 연결
  5. 브라우저에서 위 주소 접속
  6. PAUSE / RESUME / CHARGE 버튼으로 제어

----------------------------------------------------

[ 버튼 기능 ]

  PAUSE  (빨간색) : 로봇 즉시 일시정지
  RESUME (초록색) : 로봇 재개
  CHARGE (파란색) : 로봇을 충전소로 이동

[ RCS 서버 정보 ]

  IP      : 10.0.4.104
  Port    : 7000
  Robot ID: 0001
